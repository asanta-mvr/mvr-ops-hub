# web_apps_examples

WebP mockups / screenshots of other web applications kept as **design inspiration**. Not part of the build, not loaded at runtime, not deployed.

## Status

- **Untracked in git** (this directory and its `.webp` files are intentionally not versioned — see `.gitignore`).
- Only this `README.md` is committed, so the folder's purpose is documented even when it is empty.
- Safe to delete the `.webp` files anytime you finish consulting them.

## Convention

Drop reference screenshots here while exploring UI ideas. When a pattern lands in the app, the inspiration file can go.
